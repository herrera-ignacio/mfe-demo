# MFE Demo

Demo of MFEs using Webpack Module Federation.

## Setup

1. Start _Counter_ app.
   1. `cd counter`.
   2. `yarn install`.
   3. `yarn start:live`.
   4. (Optional) You can access standalone app in `http://localhost:8081`.
2. Start _Host_ app.
   1. `cd host`.
   2. `yarn install`.
   3. `yarn start:live`.
3. Navigate to `http://localhost:8080`.
